R Statistical Modeling

Advanced Statistical Modeling and Data Analysis in R

Welcome to the R Statistical Modeling Project

This repository showcases advanced statistical modeling techniques, data analysis, machine learning, and interactive data visualization using the R programming language. Developed by Gabriel Demetrios Lafis, this project aims to provide a comprehensive platform for various statistical and data science applications.

Key Features:

  • Statistical Modeling: Regression, GLM, mixed models.
  • Time Series Analysis: ARIMA, GARCH, forecasting.
  • Machine Learning: Random Forest, SVM, clustering.
  • Interactive Visualizations: ggplot2, plotly, Shiny dashboards.
  • Automated Reporting: R Markdown for dynamic documents.
  • Statistical Tests: Hypothesis testing, ANOVA, non-parametric tests.
